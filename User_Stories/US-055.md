# US-055 · Org Admin — QR Number Visible in Admin Panel
**Epic:** 13 — Sequential QR Numbering  
**Priority:** P1  
**Status:** Open  
**Phase:** 6 (Frontend — Admin Panel QR Number Display)

---

## User Story
> As an **Org Admin**, I want to see the sequential QR number alongside each QR code entry in the admin panel, so that I can cross-reference stickers, investigate reports of specific numbers, and track which numbers belong to which batch.

---

## Acceptance Criteria
- [ ] Batch detail QR table shows `qr_number` as the first column (formatted as `No. 000012`)
- [ ] QR numbers are sortable (ascending/descending)
- [ ] QR numbers are searchable — admin can type a number to jump to that code
- [ ] User detail scan history shows the QR number of each scan
- [ ] Batch detail page shows the range of QR numbers in the batch: e.g., `QR Nos. 5,001 – 8,000`
- [ ] Export CSV (metadata.csv) includes `qr_number` column

---

## UI Changes
| Page | Change |
|------|--------|
| `BatchDetail.jsx` | Add `No.` column as first column in QR table; show number range in batch info |
| `UserDetail.jsx` | Add QR number to scan history table |
| `BatchDetail` header | Show `QR range: 5,001–8,000` in info cards |

---

## API Change
| Endpoint | Addition |
|----------|---------|
| `GET /api/batch/:id/qrcodes` | Include `qr_number` in each row |
| `GET /api/user/:id` | Include `qr_number` in scan history items |
