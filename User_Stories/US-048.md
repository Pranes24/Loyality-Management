# US-048 · Org Admin — View QR Quota Usage
**Epic:** 12 — QR Quota Management  
**Priority:** P1  
**Status:** Open  
**Phase:** 5 (Frontend — Org Admin Quota Display)

---

## User Story
> As an **Org Admin**, I want to see how many QR codes I have used out of my total allocation, so that I can plan my batch creation and know when I am running low.

---

## Background
The super admin assigns a quota (e.g., 50,000) to the org. Every QR code created across all batches draws from this pool. The org admin must be able to clearly see their remaining quota at all times — on the dashboard and on the batch creation screen.

---

## Acceptance Criteria
- [ ] Dashboard shows a quota usage bar: `Used 12,400 of 50,000 QR Codes`
- [ ] Progress bar fills proportionally (amber when >80%, red when >95%)
- [ ] Remaining count is shown numerically: `37,600 remaining`
- [ ] If quota = 0 (not yet assigned), display: _"QR quota not assigned — contact Super Admin"_ with no progress bar
- [ ] Quota summary is also shown on the batch creation form header
- [ ] Data refreshes after each batch is created (no stale count shown)

---

## UI Components
| Location | Display |
|----------|---------|
| Admin Dashboard | Quota card with progress bar + used/total + remaining |
| Batch List page | Small quota indicator in page header |
| Batch Create form | Remaining quota shown above quantity input field |
