# US-054 · Enterprise QR Sticker Design — Layout & Branding
**Epic:** 14 — Enterprise Sticker Design  
**Priority:** P1  
**Status:** Open  
**Phase:** 7 (QR Sticker Layout Update)

---

## User Story
> As an **Org Admin**, I want the QR stickers to display our organisation's branding on the left and the QR code with its number on the right, so that the stickers look professional and carry the UPI logo for user trust.

---

## Background
The sticker is printed and physically attached to products. It must look enterprise-grade. The layout is fixed:
- **Left side**: Org logo, org name, UPI logo (copyright/trust mark)
- **Right side (top)**: The QR code image
- **Right side (bottom)**: The sequential number formatted as `No. 00012`

The UPI logo establishes payment legitimacy. The number below the QR enables manual fallback (US-052).

---

## Sticker Layout (Portrait, ~5cm × 3cm)
```
┌─────────────────────────────────────┐
│  ┌──────────┐      ┌─────────────┐  │
│  │ ORG LOGO │      │             │  │
│  └──────────┘      │   QR CODE   │  │
│                    │   IMAGE     │  │
│  ORG NAME          │             │  │
│                    └─────────────┘  │
│  ━━━━━━━━━━━━      No. 000012       │
│  [UPI Logo]                         │
└─────────────────────────────────────┘
```

---

## Acceptance Criteria
- [ ] Sticker is generated as part of the QR export (ZIP download from BatchDetail)
- [ ] Each sticker is a PNG image (300 DPI for print quality)
- [ ] Left column contains: org logo (if uploaded), org name, UPI logo
- [ ] Right column contains: QR code image on top, `No. XXXXXX` (zero-padded to 6 digits) below
- [ ] If org has no logo uploaded, show org name text only on left
- [ ] Sticker dimensions: 400px × 250px at 96dpi (scales to print at 300dpi)
- [ ] Font: bold sans-serif for org name; monospace for QR number
- [ ] UPI logo included as a static asset in the backend
